## 🚀 ما هو Growth Hunters؟

<div align="center">

> **Growth Hunters هو نظام تشغيل متكامل للنمو — لا مجرد أداة تسويق.**

</div>

نحن نبني **Automated Growth Engine** مخصص لكل شركة، يعمل بشكل مستمر 24/7 بدون تدخل يدوي. النظام مبني على **n8n** كـ orchestration layer، متصل بـ **ERPNext** للإدارة، و**GPT-4o** للذكاء الاصطناعي.

```
الشركة التقليدية:                    Growth Hunters:
┌─────────────────┐                  ┌────────────────────────────────┐
│ 5 متخصصين       │       vs.        │ 1 نظام ذكي متكامل              │
│ تنسيق يدوي      │                  │ تشغيل تلقائي كامل             │
│ تقارير أسبوعية  │                  │ لوحة تحكم real-time            │
│ $15,000+/شهر    │                  │ من $799/شهر                    │
└─────────────────┘                  └────────────────────────────────┘
```

## 🤖 الـ 6 AI Agents

### `FLOW 01` — 🎯 Client Onboarding Automation

> **Trigger:** Webhook POST `/new-client` أو Manual  
> **وظيفة:** أتمتة كامل عملية استقبال العميل الجديد

```
📥 Webhook/Manual
       │
       ▼
⚙️  Set Client Variables
    ├── clientName, clientEmail, industry, budget
    └── projectId: GH-YYYYMMDD-XXXX (auto-generated)
       │
       ├──► 🔗 HubSpot CRM → Create Contact (Lifecycle: customer)
       │
       ├──► 📋 ClickUp → Phase 1 Task (Due: 7 days, Priority: High)
       │         └── 6 subtasks: Onboarding call, Audit, Competitors...
       │
       ├──► 📋 ClickUp → Phase 2 Task (Due: 21 days)
       │         └── Milestone Gate: Blueprint Sign-Off required
       │
       ├──► 📧 Welcome Email → Branded HTML + Project ID
       │
       ├──► 💬 Slack → #new-clients alert + action required
       │
       └──► 🏢 ERPNext → Create Project + Task (6-month timeline)
```

**⏱️ وقت التنفيذ:** أقل من 30 ثانية من لحظة التسجيل  
**📦 الناتج:** عميل مُعدّ بالكامل في جميع الأنظمة

---

### `FLOW 02` — ⏰ Daily Task Monitor

> **Trigger:** كل 24 ساعة تلقائياً  
> **وظيفة:** مراقبة المهام المتأخرة وإرسال تنبيهات فورية

```
⏰ Schedule (24h)
       │
       ▼
📊  Check ClickUp Tasks
    └── Filter: status=In Progress, dueDate < now+2days
       │
       ▼
🔀  Has Overdue Tasks?
    ├── YES ──► 🚨 Slack #alerts → Task list + @channel
    └── NO  ──► ✅ Silent pass (no noise)
```

**📊 SLA Tracked:** Response time < 2h لأي مهمة متأخرة

---

### `FLOW 03` — 📈 Weekly Analytics Report

> **Trigger:** كل جمعة تلقائياً  
> **وظيفة:** تقرير أداء شامل من GA4 + Meta Ads + AI Insights

```
📅 Weekly Scheduler (Friday)
       │
       ├──► 📈 Fetch GA4 Data    ──┐
       │    └── sessions, conversions, revenue    │
       │                                           ▼
       └──► 📢 Fetch Meta Ads ──► 🔀 Merge ──► 🧮 Calculate KPIs
                └── spend, ROAS, CPC, impressions          │
                                                            │
                         ┌──────────────────────────────────┤
                         │                                   │
                         ▼                                   ▼
              💬 Slack: Weekly Report          🤖 GPT-4o Analysis
              📧 HTML Email to Client          └── 3 insights + 2 actions
                                               💬 Slack: #ai-insights
```

**📊 KPIs المحسوبة:**

| Metric | Formula | Status Logic |
|--------|---------|-------------|
| ROAS | Revenue ÷ AdSpend | 🟢 ≥3x · 🟡 ≥2x · 🔴 <2x |
| CAC | AdSpend ÷ Conversions | Target: -30% |
| CVR | Conversions ÷ Sessions × 100 | Target: +50% |
